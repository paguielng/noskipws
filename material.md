# Guide des matériaux & imprimantes 3D pour le robot Freenove

| Pièce                           | Matériau recommandé | Imprimante conseillée                | Notes |
|--------------------------------|----------------------|--------------------------------------|-------|
| Châssis / Corps                | PETG                | Bambu Lab P1S / Prusa MK4           | Bon compromis solidité/légèreté |
| Pattes / Articulations         | PETG ou ABS/ASA     | Bambu Lab P1S (enclosure) / MK4     | Très sollicitées → robustesse |
| Supports de servos             | PETG                | Toutes Bambu / Prusa                | Résistance + stabilité |
| Boîtier batterie               | PETG                | Toutes Bambu / Prusa                | Supporte la chaleur légère |
| Boîtier caméra                 | PETG                | Toutes Bambu / Prusa                | Bonne précision et durabilité |
| Clips / Pièces flexibles       | TPU 95A             | Bambu Lab X1C / Prusa MK4           | Flexibilité nécessaire |
| Supports électroniques         | PLA+ ou PETG        | Toutes Bambu / Prusa                | Peu d'effort mécanique |
| Coins renforcés / pièces fortes| ABS/ASA             | Bambu P1S/X1C (enclosure) / MK4     | Très haute résistance |
